Inclusão de imagens:

O ~\LaTeX~ não consegue manipular imagens sozinho,precisamos de adicionar o pacote graphicx.
Para usa-lo basta usar \usepackage{graphicsx} no inicio do documento

O comando \graphicspath{ {./imagens/} } diz ao ~\LaTeX~ que as imagens são guardadas no diretório "imagens"
dentro do diretório do documento principal

O comando \includegraphics{"nome da imagem"} é o comando cujo inclui as imagens no documento.

Podemos observar um urso - \includegraphics[width=0.5]{urso.pdf} - no meio desta frase ,que originaria o resultado:

Aqui reparamos que urso é o nome do ficheiro que contém a imagem do urso,depois urso.pdf torna-se na imagem
Também é possivel utilizar esta imagem dentro de um objeto flutuante:

                \documentclass{article}
                \usepackage{graphicx}
                \graphicspath{ {./images/} }
                     
                \begin{document}
                    podemos incluir imagens aleatórias ao nosso gosto.
                     
                    \includegraphics{urso.pdf}
                     
                    Em cima temos uma imagem aleatória.
                \end{document}

Podem existir vários ficheiros para a mesma imagem, cada um com o seu formato, e deste modo facilita-se a compilação do documento para formatos de saída diferentes.
O comando de inclusão de imagens possui várias opções, entre as quais as que permitem redimensionar ou de outra forma ajustar a imagem a incluir:

height - altura da imagem.

width- largura da imagem.

scale - fator de escala.

angle - ângulo de rotação.
