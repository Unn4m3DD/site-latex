Existem dois tipos de listas em ~\LaTeX~, listas ordenadas e não ordenadas. Para criação de listas não ordenadas utiliza-se o comando "\begin{itemize} ... \end{itemize}" juntamente com o comando "\item":

  • Item0
  • Item1
  • Item2

Gerado com o código:
\begin{itemize}
\item item0
\item item1
\item item2
\end{itemize}

Para criação de listas ordenadas utiliza-se o comando "\begin{enumerate} ... \end{enumerate}" juntamente com o comando "\item":

  1. Item0
  2. Item1
  3. Item2

Gerado com o código:
\begin{enumerate}
\item item0
\item item1
\item item2
\end{enumerate}

Podem criar-se tabelas aninhadas que, caso numeradas, criam numeração automatica das tabelas internas com um simbolo diferente:

  1. item0
    1. item00
    2. item01
      1. item010
      2. item111
      3. item112
    4. item02
    • item020
    • item121
    • item222

Este resultado difere um pouco do obtido atravez do ~\LaTeX~ pois foi uma tentativa de replicação com HTML já que a biblioteca MathJax não suporta as listas ~\LaTeX~. Algo semelhante seria gerado pelo codigo:

\begin{enumerate}
\item item0
\begin{enumerate}
\item item00
\item item01
\begin{enumerate}
\item item010
\item item111
\item item212
\end{enumerate}
\item item02
\begin{itemize}
\item item020
\item item121
\item item222
\end{itemize}
\end{enumerate}
\item item2
\end{enumerate}
